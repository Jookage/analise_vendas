# Análise de Vendas — Projeto Pessoal (Portfólio)


## Sobre este projeto
Projeto de análise exploratória e métricas de vendas usando Python. Demonstrando na prática: ETL básico, EDA, visualizações e geração de relatório.

## Linguagens e bibliotecas usadas
- Python 3.x
- pandas, matplotlib, seaborn
- Git & GitHub

## Arquivos principais
- `analise_vendas.py` — script principal de análise
- `vendas.csv` — dataset fictício (Como um projeto básico, solicitei que uma IA criasse pra mim um CSV robusto para uso de exemplo, porém com as devidas alterações se torna funcional em qualquer dataset)
- `reports/figures/` — imagens dos gráficos
