# Análise de Vendas — Projeto Pessoal

Resumo: Projeto de análise de dados com Python. Contém `analise_vendas.py` e um dataset fictício `vendas.csv`. Gera gráficos e métricas de vendas e lucro.

## Como rodar
1. Criar virtualenv:
   `python -m venv venv`
2. Ativar: `venv\Scripts\activate` (Windows)
3. Instalar dependências:
   `pip install pandas matplotlib seaborn`
4. Rodar:
   `python analise_vendas.py`

## Estrutura
- analise_vendas.py
- vendas.csv
- reports/figures/
